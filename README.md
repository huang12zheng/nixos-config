This repository contains the Nix / NixOS configuration for all of my systems. Start from `flakes.nix` (see [Flakes](https://nixos.wiki/wiki/Flakes)).

## Platforms

### NixOS

```sh-session
make nixos
```

### macOS 

```sh-session
make darwin
```
